Covercase

A slider menu with vertical or horizontal orientation

View project on GitHub

covercase

It's a ultra light weight jQuery plugin that that turns the first level children of an element into a slider menu. The orientation can be set to horizontal or vertical.
There is no auto-scroll ability, this is not a carrousel
You can scroll inside it with a mouse wheel, arrows, or swipe on touch screen.
Children elements can be images, links, div....

The vertical orientation example is located in the right side bar -------------->
